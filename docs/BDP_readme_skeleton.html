BrainSuite Diffusion Pipeline (BDP) enables fusion of diffusion and structural MRI information for advanced image and connectivity analysis. It provides various methods for distortion correction, co-registration, diffusion modeling (DTI and ODF) and basic ROI-wise statistic. BDP is a very flexible command line tool and supports diverse diffusion data-acquisition and data-processing approaches.

It is highly recommended for users to go through description of all the flags which is described in the table below. Flags are braodly classified in sub-sections for easy parsing. For more information and updates, please see http://brainsuite.org/processing/diffusion/.

BDP is released under (see license tab for details). Users can find details of their custom processing and full list of reference papers which should be cited in *.Summary.txt file. Please cite at least following papers if you use BDP in your own work.

  • C Bhushan, JP Haldar, S Choi, AA Joshi, DW Shattuck, RM Leahy, "Co-registration and distortion correction of diffusion and anatomical images based on inverse contrast normalization." NeuroImage, (2015), DOI: 10.1016/j.neuroimage.2015.03.050.

  • JH Kim, JP Haldar, Z-P Liang, S-K Song, "Diffusion tensor imaging of mouse brain stem and cervical spinal cord", Journal of Neuroscience Methods, Vol. 176, Issue 2, Pages 186-191, (2009), DOI: 10.1016/j.jneumeth.2008.09.005.
